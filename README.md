# Aplicativo de controle financeiro

Esse app foi construído para que o usuário tenha um controle dos gastos mensais. 

Toda estrutura dele é em React Native. 

Foi utilizado o Expo para testar o aplicativo. 

# Bibliotecas utilizadas

Moti - https://moti.fyi/ 

Essa é uma biblioteca para animações em React Native. 

React Native Reanimated - https://docs.swmansion.com/react-native-reanimated/
